RAPP Stack Cubby

Local development product. Nothing on this page is a publication, release, live attestation, or running service.

The repository is the product: it carries the normalized context, runtime, actual agents, controller, schemas, tests, dependency locks, packaging tools, hatch tools, and operations needed for a handoff.

Artifact path

source repository → rapp-cubby/1.0 → rapp-application/1.0 → Store ZIP and brainstem-egg/2.3-cubby → rapp-installed-twin/1.0.

Safety boundary

Generated keys, messages, enrollment data, journals, and installed state stay outside source and artifacts. Hatch verifies all bytes before extraction and does not start the product.

Handoff
